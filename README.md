Fitness Vlog Website
A modern and responsive fitness vlog website built using HTML, CSS, and JavaScript. The website shares workout tips, fitness blogs, diet plans, and healthy lifestyle content with an interactive and user-friendly design.
Features
Responsive design for all devices
Interactive navigation menu
Fitness blog and workout sections
Smooth scrolling and animations
Form validation using JavaScript
Modern UI with CSS styling
Technologies Used
HTML
CSS
JavaScript
Purpose
This project demonstrates front-end web development skills by creating a visually appealing and interactive fitness-related website.
